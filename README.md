# Tactical Monitor // GEO-SENTINEL

Strategic 24/7 Command Center for Intelligence Analysis and Global Event Monitoring.

## Project Structure

This is a monorepo consisting of:
- `geo-sentinel/`: A Next.js 15 tactical dashboard featuring live mapping, intelligence feeds, and visual surveillance.
- `data-processing/`: A FastAPI backend for intelligence extraction and real-time data streaming.

## Features

- **Live Intelligence Feed**: Real-time aggregation of global strategic events.
- **Tactical Mapping**: Interactive React-Leaflet map with asset tracking, coordinate jittering, and threat-level animations.
- **Surveillance Hub**: Tabbed YouTube Live integration for City Surveillance and Global News.
- **DEFCON Status**: Dynamic escalation barometer synced with sovereign threat levels.
- **Executive Summary**: Strategic briefings on tactical hotspots and energy market impacts.

## Setup Instructions

### Frontend (Geo-Sentinel)
1. Navigate to `geo-sentinel/`
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Access at `http://localhost:3000`

### Backend (Data Processing)
1. Navigate to `data-processing/`
2. Install dependencies: `pip install -r requirements.txt`
3. Run FastAPI application: `python -m uvicorn main:app --reload`
4. API endpoint: `http://localhost:8000/data`

## Technologies

- **Frontend**: Next.js 15, React, Tailwind CSS, Framer Motion, Lucide React, React-Leaflet.
- **Backend**: Python, FastAPI, Uvicorn.
- **Aesthetics**: Custom Retro-Tactical UI with Neon Glow and Scanline overlays.
